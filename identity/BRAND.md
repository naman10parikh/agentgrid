# Lattice — Brand

## Name Rationale

**Lattice** — a lattice is a regular, interlocking grid structure. The word captures the core product literally (NxM panes) and symbolically (agents woven together into one coherent fabric). It is one word, geometric, and memorable.

## Tagline

> NxM agents. One command. Total visibility.

## Colors

Warm black (#141312) background · Amber accent (#f59e0b) for active panes · Zinc-400 for idle · Green-400 for done states. Poppins for UI labels, monospace for pane output.

## Landing Copy

Lattice turns your terminal into a command center. One `agentgrid 2x3` command and six Claude Code agents appear in front of you — each named, each focused, each visible. No background sessions. No mystery. The chairman watches the work happen.
